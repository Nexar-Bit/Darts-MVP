This is a [Next.js](https://nextjs.org) project bootstrapped with TypeScript, Tailwind CSS, and additional dependencies.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Dependencies

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **@supabase/supabase-js** - Supabase client for authentication
- **@stripe/stripe-js & stripe** - Stripe payments
- **lucide-react** - Icon library
- **date-fns** - Date formatting utilities

## Project Structure

- `/app` - Next.js App Router directory
- `/components` - React components
- `/public` - Static assets
